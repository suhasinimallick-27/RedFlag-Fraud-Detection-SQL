# RedFlag Fraud Detection using SQL

## Project Overview
This project analyzes financial transaction data to detect 12 different fraud patterns using SQL.

## Fraud Patterns Detected
- Velocity Fraud
- Card Testing
- Merchant Collusion
- Geographic Impossibility
- Velocity Spike
- Failed Transaction Abuse
- Credit Transaction Abuse
- Round Amount Transactions
- Threshold Transactions
- Monthly Transaction Spike
- Money Laundering Patterns
- Suspicious User Behaviour

## Tech Stack
- MySQL
- SQL
- Window Functions
- Common Table Expressions (CTEs)
